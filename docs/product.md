# FoldForge Product Overview

## Vision

FoldForge helps ordinary users — hobbyists, parents, educators — turn ideas into **actually buildable** papercraft kits, not just unfolded meshes.

## MVP Scope

1. Upload 3D model (OBJ / STL / GLB / GLTF)
2. Preview in browser
3. Auto clean, decimate, unfold
4. Add tabs, numbers, fold/cut lines
5. Layout on A4 / A3 / Letter
6. Export PDF, SVG, ZIP

## Difficulty Levels

| Level | Target |
|-------|--------|
| Easy | Low face count, large parts |
| Standard | Balanced detail |
| Advanced | Higher detail, more pieces |

## Styles (MVP UI)

- Low Poly
- Cute / Chibi
- Geometric Sculpture

Style affects simplification presets in later steps; MVP focuses on upload pipeline first.

## Out of Scope (MVP)

- Text-to-3D
- Image-to-3D
- User accounts
- Community sharing
- Assembly instruction PDF
